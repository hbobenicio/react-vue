# react-vue

> Um exemplo de setup Webpack com React e Vue

![Screenshot](docs/img/ss.png)

### Instalar as dependências

`npm install`

### Para rodar o servidor de desenvolvimento

```
npm run dev
```

### Build

Desenvolvimento:

```
npm run build:dev
```

Produção:

```
npm run build:prod
```
